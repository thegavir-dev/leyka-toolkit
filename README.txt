Дополнение к плагину Leyka: чекбокс подписки на новости в форме пожертвования с автоматической меткой донора.

== Описание ==

Leyka Toolkit добавляет чекбокс «Подписаться на новости» в формы пожертвований Leyka. При успешной оплате донору автоматически присваивается настраиваемая метка (тег).

= Возможности =

* Чекбокс подписки в формах Leyka (шаблоны need-help, star, revo и другие)
* Автоматическое присвоение метки донору после оплаты
* Настраиваемый текст чекбокса и имя метки
* Опция «отмечен по умолчанию»
* Встроенный лог для диагностики
* Поддержка как WP users, так и leyka_donor post type

= Требования =

* WordPress 5.6+
* PHP 7.4+
* Плагин [Leyka](https://wordpress.org/plugins/leyka/) (активный)
* Включённая функция «Управление донорами» в настройках Leyka

== Установка ==

1. Загрузите папку `leyka-toolkit` в `/wp-content/plugins/`.
2. Активируйте плагин в разделе «Плагины».
3. Убедитесь, что плагин Leyka установлен и активен.
4. Убедитесь, что в настройках Leyka включено «Управление донорами».
5. Перейдите в меню «Leyka Toolkit» → «Настройки» и задайте параметры.

== Настройки ==

* **Включить чекбокс** — показывать/скрывать чекбокс в формах.
* **Текст чекбокса** — текст рядом с чекбоксом (по умолчанию: «Подписаться на новости»).
* **Отмечен по умолчанию** — чекбокс будет отмечен при загрузке формы.
* **Метка донора** — имя тега, который присваивается донору (по умолчанию: `newsletter`).

== Диагностика ==

Перейдите в «Leyka Toolkit» → «Лог», включите логирование и проведите тестовое пожертвование. Лог покажет каждый шаг: сохранение подписки, поиск донора, создание и назначение метки. Файл лога: `wp-content/uploads/leyka-toolkit/toolkit.log`.

== Часто задаваемые вопросы ==

= Метка не появляется у донора =

1. Убедитесь, что в Leyka включено «Управление донорами» (Leyka → Настройки → Доноры и аккаунты).
2. Включите лог в «Leyka Toolkit» → «Лог» и проведите тестовый платёж.
3. Проверьте лог — он покажет, на каком шаге произошёл сбой.

= Чекбокс не отображается =

1. Проверьте, что чекбокс включён в настройках плагина.
2. Убедитесь, что используемый шаблон Leyka поддерживается (need-help, star, revo). Для других шаблонов используйте фильтр `leyka_toolkit_supported_templates`.

= Как добавить поддержку кастомного шаблона? =

Добавьте в `functions.php` темы:

`add_filter('leyka_toolkit_supported_templates', function ($templates) {
    $templates[] = 'my-template';
    return $templates;
});`
