Дополнение к плагину Leyka: добавляет чекбокс подписки в форму пожертвования и автоматически присваивает донору метку после успешной оплаты.

== Описание ==

Leyka Toolkit расширяет формы пожертвований Leyka простым чекбоксом «Подписаться на новости». Если донор оставил галочку и платёж прошёл успешно, плагин назначает ему указанную метку.

Плагин подойдёт, если нужно собирать согласие на подписку прямо в форме пожертвования и не терять эту информацию после оплаты.

= Что умеет =

* Добавляет чекбокс подписки в формы Leyka
* Работает с шаблонами `need-help`, `star`, `revo` и другими через фильтр
* Автоматически присваивает донору метку после успешного пожертвования
* Позволяет настроить текст чекбокса и имя метки
* Поддерживает чекбокс, отмеченный по умолчанию
* Ведёт лог для диагностики
* Работает как с `WP users`, так и с `leyka_donor`

= Требования =

* WordPress 5.6 или выше
* PHP 7.4 или выше
* Установленный и активный плагин [Leyka](https://wordpress.org/plugins/leyka/)
* Включённая опция «Управление донорами» в настройках Leyka

== Установка ==

1. Загрузите папку `leyka-toolkit` в каталог `/wp-content/plugins/`.
2. Активируйте плагин в разделе «Плагины».
3. Убедитесь, что Leyka установлен и активен.
4. Проверьте, что в настройках Leyka включено «Управление донорами».
5. Откройте «Leyka Toolkit» -> «Настройки» и задайте нужные параметры.

== Настройки ==

* **Включить чекбокс** - показывает или скрывает чекбокс в форме
* **Текст чекбокса** - текст рядом с чекбоксом
* **Отмечен по умолчанию** - включает предустановленную галочку при загрузке формы
* **Метка донора** - имя тега, который будет назначен донору после успешной оплаты

== Диагностика ==

В разделе «Leyka Toolkit» -> «Лог» можно включить логирование и провести тестовое пожертвование. Лог помогает понять, что произошло на каждом этапе: сохранилось ли согласие, нашёлся ли донор, была ли создана и назначена метка.

Файл лога: `wp-content/uploads/leyka-toolkit/toolkit.log`.

== Часто задаваемые вопросы ==

= Метка не назначается =

1. Убедитесь, что в Leyka включено «Управление донорами» (`Leyka -> Настройки -> Доноры и аккаунты`).
2. Включите логирование в разделе «Leyka Toolkit» -> «Лог».
3. Выполните тестовый платёж и проверьте лог: там будет видно, на каком шаге возникла проблема.

= Чекбокс не отображается =

1. Проверьте, что чекбокс включён в настройках плагина.
2. Убедитесь, что используется поддерживаемый шаблон Leyka: `need-help`, `star`, `revo`.
3. Если у вас свой шаблон, добавьте его через фильтр `leyka_toolkit_supported_templates`.

= Как добавить поддержку своего шаблона? =

Добавьте в `functions.php` темы:

```php
add_filter('leyka_toolkit_supported_templates', function ($templates) {
    $templates[] = 'my-template';
    return $templates;
});
```
