Leyka Toolkit 0.3.12

WordPress-плагин для расширения форм пожертвований Leyka.

Возможности:
- чекбокс «Подписаться на новости»;
- сохранение `leyka_donor_subscribed` в meta донации;
- назначение метки донору после успешного пожертвования;
- чекбокс согласия на ежемесячное автоматическое списание для рекуррентных пожертвований;
- вставка рекуррентного согласия внутрь native agreement span Leyka через `display: contents`;
- порядок галок: штатные соглашения Leyka, рекуррентное согласие, подписка;
- выравнивание вертикальных отступов рекуррентной галки и подписки со штатными соглашениями Leyka;
- подсказка рекуррентного согласия под кнопкой пожертвования;
- собственная фронтовая валидация обязательного рекуррентного согласия на уровне кнопки, click и submit;
- поддержка рекуррентного переключателя как через `.leyka-recurring`, так и через `data-periodicity` + `leyka_recurring`;
- сохранение `leyka_recurring_agreed` в meta донации;
- назначение отдельной метки донору за согласие на рекуррент, по умолчанию `recurring-agree`;
- настройки текстов и обязательности в админке;
- служебное логирование.

Требования:
- WordPress;
- активный плагин Leyka;
- Leyka 3.20 или выше для сценариев рекуррентного согласия по ТЗ.
